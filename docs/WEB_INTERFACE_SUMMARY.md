# Web Interface Summary

> **DEPRECATED**: The Buddi project has pivoted away from a clinician-facing chat dashboard to a purely invisible RCM (Revenue Cycle Management) and compliance tool. The frontend web interface is officially discontinued to avoid "tab fatigue" and directly address the enterprise market requirements.

Please refer to the `README.md` and `FRONTEND_BACKEND_CONNECTION.md` (now the API integration guide) for the current architecture of the Buddi Agent.
